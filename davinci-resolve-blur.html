DaVinci Resolve Blur Guide

Complete guide to blurring regions in DaVinci Resolve using Fusion

What is Fusion?

Fusion is DaVinci Resolve's node-based compositing and VFX environment. It's integrated directly into Resolve and provides professional-grade tools for motion graphics, visual effects, and compositing. Unlike layer-based systems, Fusion uses a node graph where each effect is a "node" that can be connected to create complex effect chains.

Why Use Fusion for Blurring?

While DaVinci Resolve offers multiple ways to blur content, Fusion is the most powerful and flexible option for precise, region-specific blurring:

Edit Page Blur (Basic)

  • Method: OpenFX blur effect in Inspector
  • Pros: Quick, simple interface
  • Cons: Limited masking options
  • Use case: Full-frame blur, simple vignettes

Fusion Page Blur (Advanced)

  • Method: Node-based compositing
  • Pros: Precise masks, multiple regions, animation
  • Cons: Steeper learning curve
  • Use case: Complex selective blur, tracked objects

Complete Fusion Blur Workflow

1

Import and Setup Media

Before you can work in Fusion, you need to get your media into DaVinci Resolve:

  • Open DaVinci Resolve and create a new project (or open existing)
  • Navigate to the Media Pool (top-left in Edit page)
  • Drag and drop your image or video file into the Media Pool
  • Drag the media from Media Pool to the timeline
2

Switch to Fusion Page

Access the Fusion compositing environment:

  • Click the Fusion tab at the bottom of the screen
  • You'll see a node graph area (initially showing MediaIn → MediaOut)
  • The MediaIn node represents your source clip
  • The MediaOut node sends the result back to the timeline

Understanding the Node Graph

Everything in Fusion flows from left to right. The MediaIn node is your source, and MediaOut is your result. Any nodes you add in between will process the image before it returns to the timeline. Think of it like a signal flow diagram.

3

Add a Blur Node

Insert the blur effect into your node graph:

  • Select the MediaIn node (click on it)
  • In the toolbar, click Effects → Blur → Blur
  • Or use the shortcut: Shift + Space, type "blur", press Enter
  • The Blur node will appear and auto-connect: MediaIn → Blur → MediaOut
MediaIn → Blur → MediaOut
4

Adjust Blur Settings

Configure the blur intensity in the Inspector panel (top-right):

  • Blur Size: Controls the blur radius (0-100+). Start with 10-20 for subtle blur, 50+ for heavy obscuring
  • Blend: Controls opacity of blur effect (1.0 = full blur)
  • Filter Type: Leave as "Gaussian" for natural blur (other options: Box, Soften, Defocus)

Current Issue: Full-Frame Blur

At this point, the entire image is blurred. This is not what we want for selective region blurring. We need to add a mask to limit the blur to specific areas. Continue to the next step.

5

Add a Rectangle Mask

Create a mask to define the blur region:

  • Select the Blur node
  • In the toolbar: Effects → Mask → Rectangle
  • Or press Shift + Space, type "rectangle", Enter
  • A Rectangle node will be added and connected to the Blur's mask input (blue connection)
MediaIn → Blur → MediaOut
Rectangle (mask) → Blur.EffectMask
6

Position and Scale the Mask

Define where the blur should appear:

  • Select the Rectangle node
  • In the Viewer (center area), you'll see a rectangle overlay
  • Move: Click and drag the center circle to reposition
  • Scale: Drag corner handles to resize
  • Rotate: Drag the rotation handle (if needed)
  • In the Inspector, you can numerically adjust:
    • Center - X/Y position (0.5, 0.5 = center)
    • Width & Height - Size of rectangle
    • Corner Radius - Rounded corners (0 = sharp, 1 = circle)
    • Soft Edge - Feather the mask edge (0.1-0.3 recommended for natural blend)

Pro Tip: Soft Edges

Always add a Soft Edge value (0.1 or higher) to your masks. This creates a gradual transition between blurred and sharp areas, making the effect look natural rather than cut-out and obvious.

7

Preview and Refine

Fine-tune your blur effect:

  • The Viewer shows a live preview of your blurred region
  • Select the Blur node to adjust blur intensity
  • Select the Rectangle node to adjust mask position/size
  • Use timeline playback to preview on video clips

Adding Multiple Blur Regions

To blur multiple areas in the same frame, you need to merge multiple blur operations:

1

Duplicate the Blur Chain

  • Select the Blur node and Rectangle node (Ctrl+Click to multi-select)
  • Press Ctrl+C to copy, then Ctrl+V to paste
  • You now have a second Blur + Rectangle set
2

Add a Merge Node

  • Press Shift+Space, type "merge", press Enter
  • Connect MediaIn → Merge.Background (yellow input)
  • Connect first Blur → Merge.Foreground (green input)
  • Connect second Blur to another Merge node, then chain them
  • Connect final Merge → MediaOut
MediaIn → Merge1.BG
Blur1 → Merge1.FG → Merge2.BG
Blur2 → Merge2.FG → MediaOut

Complexity Warning

As you can see, adding multiple blur regions in Fusion quickly becomes complex with nested Merge nodes. For static images with multiple regions, the Browser Blur Tool is significantly faster and easier. Reserve Fusion for video clips or animated effects.

Advanced Techniques

Animated Blur (Tracking Movement)

To blur an object that moves across the frame:

  1. Add a Planar Tracker node before your mask
  2. Track the object in the Tracker controls
  3. Connect Tracker → Rectangle (or other mask)
  4. The mask will follow the tracked object automatically

Different Blur Types

Fusion offers multiple blur algorithms:

  • Gaussian Blur: Standard smooth blur (most common)
  • Defocus: Simulates camera lens defocus (bokeh effect)
  • Directional Blur: Motion blur in specific direction
  • Radial Blur: Blur radiating from center point (zoom/spin effect)
  • Soften: Gentle blur preserving edge detail

Pixelate Effect (Alternative to Blur)

For a more aggressive obscuring effect:

  1. Instead of Blur node, add a Grid Warp node
  2. Or use Effects → Stylize → Pixelate (if available)
  3. Reduce resolution to create blocky pixelation
  4. Better for hiding text/sensitive data than blur

Common Issues and Solutions

Blur Not Appearing

Problem: Blur node added but no blur visible

Solution: Check that Blur Size is high enough (try 20+) and that nodes are properly connected (MediaIn → Blur → MediaOut)

Entire Image is Blurred

Problem: Can't limit blur to specific region

Solution: Make sure you've added and connected a mask node (Rectangle/Ellipse) to the Blur node's blue EffectMask input

Mask Edge Too Hard

Problem: Blur region has sharp, obvious edges

Solution: Select the mask node (Rectangle/Ellipse) and increase the Soft Edge parameter to 0.2 or higher

Performance Slow

Problem: Fusion preview is laggy with blur applied

Solution: Lower the timeline playback resolution (top-right viewer dropdown: change from "Full" to "Half" or "Quarter")

Exporting Blurred Content

Once you're satisfied with your blur:

  1. Return to the Edit page
  2. Your blurred clip appears in the timeline with a Fusion icon
  3. To export: Click Deliver page (bottom tabs)
  4. Choose preset (e.g., "YouTube 1080p") or custom settings
  5. Click Add to Render Queue
  6. Click Render All

Alternative: Export Still Frame

For a single blurred image (not video), right-click the clip in the timeline → Generate Still → Choose format (PNG recommended). The blurred frame will be saved to your chosen location.

When to Use Browser Tool Instead

Consider using the Browser Blur Tool instead of Fusion when:

  • You're working with static images (screenshots, photos)
  • You need to blur multiple regions quickly without complex node graphs
  • You want instant export without rendering time
  • You're preparing images before importing to video editor
  • You don't need animation or tracking

Use Fusion when you need:

  • Video clip blur (multiple frames)
  • Animated blur (changing position/intensity over time)
  • Tracked blur (following moving objects)
  • Advanced blur types (defocus, directional, radial)
  • Integration within a larger Resolve project

Summary: Fusion Blur Workflow

  1. Import media to timeline
  2. Switch to Fusion page
  3. Add Blur node (Shift+Space → "blur")
  4. Add Rectangle mask node to Blur
  5. Position and scale mask in Viewer
  6. Adjust Blur Size and mask Soft Edge
  7. For multiple regions: duplicate Blur+Mask, use Merge nodes
  8. Return to Edit page, export via Deliver page
