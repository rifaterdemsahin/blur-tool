# B-Roll Blur Tool

Professional blur tool for video production workflows - quickly blur sensitive information in B-roll footage screenshots.

## Live Demo

**[Launch Tool](https://rifaterdemsahin.github.io/blur-tool/)**

## Features

- **Drag & Drop Interface** - Load images instantly
- **Multiple Blur Regions** - Apply blur to multiple areas before export
- **Gaussian Blur & Pixelate** - Two effect types for different use cases
- **Adjustable Intensity** - 1-20px blur/pixelate strength
- **Instant Export** - Export as PNG with `_blurred` suffix
- **Zero Dependencies** - Pure HTML/CSS/JS, no installation required

## Quick Start

1. Open `blur-tool.html` in your browser
2. Drag an image onto the canvas
3. Draw rectangles over areas to blur
4. Adjust intensity slider (optional)
5. Click **Export PNG**
